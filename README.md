# Python MatchCase Toolkit

## 📘 Overview

**Python MatchCase Toolkit** is a collection of beginner-friendly Python programs that demonstrate the use of the **`match-case`** statement, introduced in Python 3.10.

These programs illustrate how pattern matching can simplify decision-making by replacing multiple `if-elif-else` statements with cleaner and more readable code.

## 📂 Programs Included

### 🔹 Operator Matcher

Identifies the arithmetic operation based on the operator entered by the user.

**Supported Operators**

* `+` → Addition
* `-` → Subtraction
* `*` → Multiplication
* `/` → Division
* Default → Cannot be determined

---

### 🔹 Day Type Checker

Determines whether the entered day number represents a **Weekday** or a **Weekend**.

**Classification**

* `1` or `7` → Weekend
* `2` to `6` → Weekday
* Any other value → Invalid

---

### 🔹 Vowel and Consonant Checker

Checks whether the entered alphabet is a **Vowel** or a **Consonant**.

**Features**

* Supports both uppercase and lowercase vowels.
* Uses grouped cases with the `|` (OR) operator.

## 🛠️ Concepts Covered

* Python `match-case` Statement
* Pattern Matching
* User Input
* Selection Control Statements
* OR (`|`) Operator in Match Cases
* Default Case (`case _`)
